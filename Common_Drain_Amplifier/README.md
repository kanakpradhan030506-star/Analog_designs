# Common Drain Amplifier (Source Follower)

## Objective
To design, layout, and simulate a Common Drain Amplifier using CMOS technology and analyze its AC frequency response.

## Tools Used
- Electric VLSI
- LTspice

## Project Structure

Common_Drain_Amplifier/
├── layout/
├── schematic/
├── simulation/
└── description/

## Files

### Layout
Contains Electric VLSI layout files and screenshots.

### Schematic
Contains transistor-level schematic screenshots.

### Simulation
Contains LTspice simulation files, SPICE netlist, and AC response waveform.

### Description
Contains detailed explanation of circuit operation.

## Results
The Common Drain Amplifier was simulated using AC analysis to observe gain and phase variation across frequency.

## Applications
- Buffer circuits
- Voltage followers
- Impedance matching
- Analog integrated circuits
