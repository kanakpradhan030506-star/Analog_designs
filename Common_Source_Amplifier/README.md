# CMOS Common Source Amplifier

This project demonstrates the design and simulation of a **CMOS Common Source Amplifier** using PMOS and NMOS transistors.

## Objective

Design and analyze a CMOS common source amplifier.

## Components Used

- PMOS Transistor (W = 5)  
- NMOS Transistor (W = 20) 
- VDD = 5V  
- Vin (Input)  
- Vout (Output)  

## Circuit Working

- Input applied at gate terminal  
- Output taken from drain node  
- PMOS acts as active load  
- NMOS controls amplification  

## Simulation

DC Sweep Analysis performed from:
**0V → 5V**
Observation:
- NMOS turns ON as input increases  
- Output voltage changes accordingly  
- Proper amplifier behavior observed  

## Layout

Layout designed using CMOS VLSI rules.
Layers used:
- Diffusion  
- Polysilicon  
- Metal  
- Contacts  

## Applications

- Analog Amplifiers  
- CMOS Integrated Circuits  
- Signal Processing  
- VLSI Design
